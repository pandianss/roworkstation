# User Guide

This guide provides an overview of the RO Workstation features and how to use them effectively for Regional Office operations.

## Dashboard

The **Dashboard** is the main entry point, providing a high-density view of real-time MIS KPIs, growth trends, and pending tasks. It serves as a command center for regional management.

## MIS Analytics

The **MIS** module allows you to:
- View key performance indicators (KPIs) for the region.
- Analyze growth trends across different business segments.
- Export data to Excel for further offline analysis.

## Document Centre

The **Document Centre** is a centralized hub for all document generation activities.

### Office Note Hub
A robust, audit-ready system for managing Office Notes:
- **CRUD Operations**: Create, Read, Update, and Delete office notes.
- **Sequential Referencing**: Automated management of reference numbers.
- **PDF Generation**: Regenerate professional PDF versions of any office note.

### High Value DD Approval
A dedicated wizard for processing High Value Demand Draft approvals:
- Multi-step intake form.
- Automated branch name lookup.
- Real-time document generation.

### Letter Generator
Generate various official letters (performance recognition, communication, etc.) using unified templates.

## Branch & Campaign Management

### Branch Portal & Visits
Track and manage branch interactions:
- Log branch visits and findings.
- View branch performance profiles.
- Coordinate with branch managers.

### Campaign Tracking
Monitor the progress of regional campaigns:
- Track targets and achievements.
- Identify underperforming branches.
- Communicate campaign updates.

## Knowledge Base

The **Knowledge** module provides a central repository for policy documents, circulars, and SOPs:
- **Global Search**: Find information across all indexed documents.
- **AI-Powered QA**: Ask questions to get summaries or specific details from documents (requires Ollama).

## Statutory Returns (DICGC)

Manage compliance and statutory returns:
- Prepare and validate DICGC data.
- Ensure accuracy for audit and regulatory submission.

## Operational Wizards

A set of specialized wizards for common administrative tasks, designed to streamline complex workflows into simple, step-by-step processes.
